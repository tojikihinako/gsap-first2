インタビュー

佐野さんにSDについていろいろ質問してきました！

高校生から大学生になって...？

戸敷 : 高校から大学に入ってどう変わりましたか。

佐野 : 高校時代は、土曜日まで学校があったので日曜日しか休みがなくって、
     あんまり遊ぶこととかバイトがしっかりできなかったとこがいちばん変わったと思います。

戸敷 : その分ほかになにかしてるんですか？

佐野 : 大学に入って全休だったり土日がフリーになったことでいっぱいお金を稼いだり、 自分の好きなことができるようになりました。

戸敷 : ありがとうございます。

SDに入ってみて...？

戸敷 : SDに入って大変なことは何ですか。

佐野 : 1番大変だったなって思ったのは、課題とかなんですけど、
     特に、他の学部のみんなは楽してるというか忙しくなさそうなのに
     自分たちだけ忙しいっていうのが自分的には1番しんどいなと思いました。

戸敷 : SDに入ってよかったと思いますか。

佐野 : 入ってよかったと思います。

戸敷 : どんなとこがよかったと思いますか

佐野 : さっき言ったみたいにSDは大変な学科だし、課題もとってもあると思うけど、
     作ってる途中は何作ってるかわかんなくても、結局出来上がった時や発表した時には頑張ったなと思うし、
     そういう達成感が授業だけで得られるっていうのは他の学部とは違うところだと思います。
     授業内でいろんなことが完結できるのはとてもいいことだと思います。

戸敷 : ありがとうございます。

SDの授業は...？

戸敷 : 大変な授業は何でしたか。

佐野 : やっぱり1番大変だったのは、基礎デザイン制作の授業だと思っています。

戸敷 : 基礎デザイン制作のどんなとこが大変でしたか？

佐野 : 他の授業はもし課題が間に合わなくても自分が責任をとるだけだったりすると思うんですけど、
     基礎デザイン制作の授業はもし未完成なものであってもみんなの前で見せなきゃいけないので、そういうプレッシャーもあるし、
     あとはグループ制作とかもあったので、自分だけの責任じゃなくなったところがあったので、
     それが1番しんどい授業だなと思います。

戸敷 : ありがとうございました。
